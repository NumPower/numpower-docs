# Installing Autograd

NumPower Autograd is available through composer and uses the NumPower extension and
its powerful NDArray object as an engine for mathematical operations.
You must first compile and install the extension in your environment before installing Autograd.

## Requirements

- NumPower Extension >= 0.5.0 (https://www.numpower.org/install/from-source)
- PHP >= 8.3
- Composer

## Installing

```
$ composer require numpower/autograd
```
