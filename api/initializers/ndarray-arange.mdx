# NDArray::arange

```php
public static function arange(scalar $stop, scalar $start = 0, scalar $step = 1): NDArray;
```
Return evenly spaced values within a given interval.

---

## Parameters

### `$start`
> **Type** `scalar`
>- Start of the interval. Default value is 0.

### `$stop`
> **Type** `scalar`
>- Stop of the interval.

### `$step`
> **Type** `scalar`
>- Step of the interval. Default value is 1.

---

## Return

> **Type** - NDArray
>- A single-precision (float32) NDArray with evenly spaced values.
