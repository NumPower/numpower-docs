# NDArray::mod

```php
public static function mod(NDArray|array|scalar $a, NDArray|array|scalar $b): NDArray|double;
```
Performs element-wise modulo operation between two arrays and returns a new array containing
the result. The modulo operation calculates the remainder after division.

---

## Notes

:::tip

#### GPU SUPPORTED
This operation is supported by GPU (VRAM) and contains a custom CUDA kernel.

:::
