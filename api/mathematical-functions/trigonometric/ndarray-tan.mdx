# NDArray::tan

```php
public static function tan(NDArray|array|scalar $array): NDArray|double;
```
Calculates the element-wise tangent of an array,
returning a new array with the tangent of each element.

---

## Notes

:::tip

#### GPU SUPPORTED
This operation is supported by GPU (VRAM) and contains a custom CUDA kernel.

:::
