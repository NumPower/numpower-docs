# NDArray::cpu

```php
public function cpu(): NDArray;
```
Copy the NDArray to the CPU for computation. If the NDArray is already in RAM, a copy will still be made.

---

## Return

> **Type** `NDArray`
>- A copy of the NDArray but stored in RAM.


---
