# NDArray::reshape

```php
public static function reshape(NDArray|array|float|int $target, array $shape): NDArray|float|int;
```
Changes the shape of the NDArray.

---

## Parameters

### `$shape`
> **Type** `array`
>- The new shape of the NDArray.

---

## Return

> **Type** - NDArray
>- Return a view of the array with shape `$shape`. To be compatible, the new shape must have the same amount of
elements as the old one.
